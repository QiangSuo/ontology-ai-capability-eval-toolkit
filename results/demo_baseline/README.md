# demo_baseline

`results/demo_baseline/` is a simulated MVP evaluation result package. It demonstrates what a completed baseline evaluation result directory can look like after five MVP tasks and automatic scoring.

This directory is a demo sample only. It is not a real AI tool evaluation result.

Intentional imperfections:

- Some core concepts and relations are missing.
- Some required attributes are incomplete.
- One weakly supported inferred concept is included.
- One evidence reference is intentionally broken.
- Human review remains pending after machine scoring.
