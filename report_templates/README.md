# report_templates

## 用途

`report_templates/` 存放标准评估报告、能力矩阵、技术附录和管理层摘要模板。

## 包含内容

后续可包含：

- 标准评估报告模板
- 能力矩阵模板
- executive summary 模板
- technical appendix 模板
- 工具画像摘要模板

## 输入

- customer profile
- task results
- scoring results
- manual review
- execution log

## 输出

- 客户现场评估报告
- 内部准入建议
- 工具组合能力画像
- 需要客户补充的权限或能力清单

## 使用规范

- 报告必须区分已测能力和未测能力。
- 报告必须列出权限限制和评分上限原因。
- 报告必须明确推荐使用场景和不推荐使用场景。
- POC 准入结论必须保留人工确认环节。
