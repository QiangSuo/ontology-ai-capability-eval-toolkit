# tools

## 用途

`tools/` 预留给后续辅助工具，例如格式转换器、报告生成器、结果归档工具和本地校验工具。

## 包含内容

后续可包含：

- 文档格式转换辅助工具
- 结果归档工具
- 报告生成工具
- 数据脱敏辅助工具
- 本地校验工具入口

## 输入

- 评估结果目录
- scoring 输出
- report template
- customer profile

## 输出

- 生成后的报告
- 转换后的标准格式文件
- 脱敏后的 benchmark 输入

## 使用规范

- 当前阶段不创建任何工具脚本。
- 后续工具必须支持离线执行。
- 工具不得默认上传或外发客户数据。
- 任何会修改或删除结果的工具都必须有明确确认机制。
